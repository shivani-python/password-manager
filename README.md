# 🔐 Password Manager (Python)

A simple GUI-based Password Manager built using Python and Tkinter.

## 📌 Features
- Generate strong passwords
- Save login credentials securely
- Search saved passwords
- Data stored using JSON

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- JSON (Data storage)

## ▶️ How to Run
1. Clone the repository
2. Run main.py
3. Start saving your passwords!

## ⚠️ Note
This project is for learning purposes. Do not store real passwords.
